Recipe Manager:

Проект для управления рецептами, ингредиентами и списком покупок.
В проекте реализованы классы:

Ingredient - ингредиент рецепта;
Recipe - рецепт блюда;
ShoppingList - список покупок;
DietaryRecipe - рецепт с диетической категорией.
